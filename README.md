# AutoRegexV0

#### 2024-03-19 ~ 2024-03-25

### 기능
문자열1, 문자열 2를 입력하면 해당 문자열들에 대한 간단한 정규표현식을 반환한다.
<br>
예) context1 = 1번출구
    context2  = 2번출구
    return = "\\d+번출구"

    context1 = 000-0000-0000
    context2 = 111-1111-2222
    return = "\\d\\d\\d-\\d\\d\\d\\d-\\d\\d\\d\\d"

### 테스트 케이스
1. 맨 앞이 변하는 문자일 때
2. 중간이 변하는 문자일 때
3. 변하는 문자가 한글일 때
4. 변하는 문자가 영어일 때
